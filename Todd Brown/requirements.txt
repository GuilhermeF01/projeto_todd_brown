# States Media - Frontend para Análise de Documentos PDF
# Requisitos e Dependências do Sistema

# =============================================================================
# FRONTEND REQUIREMENTS
# =============================================================================

# Navegadores Suportados:
# - Chrome 90+
# - Firefox 88+
# - Safari 14+
# - Edge 90+

# APIs JavaScript Necessárias:
# - Fetch API (para requisições HTTP)
# - FormData API (para upload de arquivos)
# - File API (para manipulação de arquivos)
# - LocalStorage API (para persistência de dados)
# - URL API (para manipulação de URLs)
# - Navigator.share API (opcional, para compartilhamento)
# - Navigator.clipboard API (opcional, para copiar links)

# =============================================================================
# SERVIDOR WEB REQUIREMENTS
# =============================================================================

# Qualquer servidor web estático:
# - Apache HTTP Server 2.4+
# - Nginx 1.18+
# - Node.js + Express 4.0+
# - Python HTTP Server (python -m http.server)
# - Live Server (VS Code Extension)
# - Netlify, Vercel, GitHub Pages

# =============================================================================
# N8N WEBHOOK REQUIREMENTS
# =============================================================================

# N8N Version: 1.0+
# Webhook Configuration:
# - Method: POST
# - Content-Type: multipart/form-data (para upload de arquivos)
# - CORS habilitado para o domínio do frontend

# Campos esperados no webhook:
# - file (arquivo PDF)
# - nome (string)
# - email (string)
# - telefone (string)
# - tipo_analise (string)
# - origem (string)
# - timestamp (string ISO)
# - data (JSON string com todos os dados)
